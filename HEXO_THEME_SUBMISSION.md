## Installation

### NPM (Recommended)
```bash
npm install hexo-theme-halunhaku
```

### Git Clone
```bash
git clone https://github.com/forhalunhaku/hexo-theme-halunhaku.git themes/halunhaku
```

## Configuration

```yaml
# _config.yml
theme: halunhaku

# Disable built-in highlighter
syntax_highlighter: false
highlight:
  enable: false
  hljs: false
```
